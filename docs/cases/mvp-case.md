# Minimalist ErgoDonk Zero case

For those that like the look of a bare pcb, you can get away with only a 3d printed keyplate and some rubber bumpers on the underside of the pdb. You could even get away without the key plate, but the switches will likely rock in their sockets. 

3d printing the keyplate is the easiest method. There is only 1 keyplate fdm source file and it's for the left hand. The file should be mirrored before printing it for the right hand. There are notches on the underside of the plate for the keys to clip into, which is why you should mirror instead of printing the left hand plate twice. 

You could also print the source file for the acrylic key plate, and use it as a template to cut a keyplate from cardboard.

Rubber bumpers on the bottom can be subbed out for bumps of hot glue.
